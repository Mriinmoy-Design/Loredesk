# LoreDesk

LoreDesk is a static, responsive knowledge-sharing dashboard for design schools. It preserves workshop techniques, fabrication methods, vendor recommendations, machine settings, critique insights, project processes, senior handover boxes, and failure cases.

## Run Locally

Open `index.html` directly in a browser, or serve the folder with any static server:

```bash
python3 -m http.server 4174
```

Then visit `http://127.0.0.1:4174`.

## Deploy

This project is pure HTML, TailwindCSS via CDN, CSS, and vanilla JavaScript. It can be deployed directly with GitHub Pages by publishing the repository root.
